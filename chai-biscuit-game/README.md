# ☕ Chai vs Biscuit Tic Tac Toe 🍪

A premium, claymorphism-themed Tic Tac Toe game built with React, Vite, and Framer Motion. 

## ✨ Features
- **Two Player Mode:** Play as Chai (Player 1) or Biscuit (Player 2).
- **Claymorphism Design:** Soft UI, rounded corners, and warm cafe aesthetics.
- **Animations:** Smooth piece dropping and hover transitions powered by Framer Motion.
- **Match Logic:** Automatic win detection, draw detection, and score tracking.
- **Celebration:** Confetti effect drops when a player wins!
- **Responsive:** Mobile-friendly and completely responsive game layout.

## 🛠️ Tech Stack
- **Frontend:** React, Vite
- **Styling:** Vanilla CSS (Claymorphism UI)
- **Animations:** Framer Motion, React Confetti

## 🚀 Run Locally

1. Clone the repository and navigate to this folder:
```bash
cd chai-biscuit-game
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## 🌐 Deployment
This project is pre-configured for deployment on Netlify. The `netlify.toml` and `public/_redirects` files are already included to handle SPA routing correctly!
