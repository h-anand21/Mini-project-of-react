# ⏱️ Time Sketch

A modern React-based application designed for time tracking, recording laps, and managing quick notes. 

## ✨ Features
- **Timer & Laps:** Accurately track your time and record laps for your tasks.
- **Notes Management:** Seamlessly write and manage notes alongside your ongoing timers.
- **Interactive UI:** A sidebar navigation and clean component architecture.
- **Modern Styling:** Designed using Tailwind CSS for a highly responsive and sleek interface.

## 🛠️ Tech Stack
- **Frontend:** React, Vite
- **Styling:** Tailwind CSS, PostCSS

## 🚀 Run Locally

1. Clone the repository and navigate to this folder:
```bash
cd time-sketch
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## 🌐 Deployment
This project is pre-configured for deployment on Netlify. The `netlify.toml` and `public/_redirects` files are already included to handle SPA routing and direct the base directory automatically.
